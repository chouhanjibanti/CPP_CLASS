Tokens in c :- 

Tokens are the smallest block/unit of c program.


There are diff types of tokens:- 
1. Keywords 
2. Identifer
3. String 
4. Operators 
5. Special Symbols 


1. Keywords 
 :- keywords are the reserved words -> already defined in c langauge.
 you can not use keyword as a variable.

 Example :- 
 int , float , while , swith , if , else , return , void , break , continue

 int age = 20;   ✔ poss
 int if = 20;   ❌ not possible


 2. Identifier :- Name given by the programmer/ developer.

 Example :- Variable , function , Arrays 
Variable:-

  3. String :- Combination of characters/letters.

  1. Arithmetic :- + , - ,  * , / , %

  3. Logical :- AND && , OR || , not !
  4. Bitwise :- 
  5. Increment /Decrement :- x++ , ++x, x-- , --X
  6. Assignment :- += , sum = sum +i;   -> sum += i;
