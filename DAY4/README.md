## Array can be defined statically and dynamically

Static Allocation :- 
when the elements of the array are declared at compile time.
this is called static memory allocation 

int arr[] = {1,2,3,4,4,5};
-----------------
Dynamically Allocation :- 
In dynamic allocation , array element are given at  run time by the user.
'
Example :- 
int arr[5];
