Diff diff types of Loop :- 
For Loop 
While Loop
Do while loop



For Loop  and While loop:- 
1. when number of iteration is known.
1. In while number of Iteration is unknown.

2. In For loop init , condition and increment/decrement in single line.
2. IN while loop init , condition and ince/decre in diff line.

3. Syntax :- 
for(init ; condition; incre/decre){
   // statement 
}

3. Syntax :- 
while(condition){
    // statement 
    incre/decre;
}



While and do while 
1. while loop first check the condition then run.
1. do while atleast one time run before checking the condition.

2. while loop is entry control loop.
2. do while is exit control loop.

3. Syntax :- 
while(condition){
    // statement 
    incre/decre;
}

3. Syntax :- 
do{
    // statement 
    incre/decre
